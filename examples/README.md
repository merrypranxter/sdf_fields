# Examples

Working shader examples demonstrating sdf_fields recipes.

## Files

### `smooth_blobs.glsl`
Standalone ShaderToy shader: animated smooth-union metaballs.

### `gyroid_sculpture.glsl`
Standalone ShaderToy shader: gyroid bounded by a sphere with studio lighting.

Paste into [ShaderToy](https://www.shadertoy.com/new) as the Image shader.
